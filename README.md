# makeMedia
YT-DLP Downloader for TeleCero

Over 100 Links in TeleCero

## Warning
Please add your python bin folder to your $PATH for this to work.

## Playlist 2.0 Format
```
Channel Name|Channel URL
```
## Examples
### Mitele
